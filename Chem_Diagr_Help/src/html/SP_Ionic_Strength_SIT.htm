SED & PREDOM / Ionic Strength

The Specific Ion interaction Theory (SIT) was adopted by the OECD-NEA thermodynamic database project, and described in its published reviews and books, see list of references.

In the SIT the activity coefficient of an ion “i” with electric charge Zi is given by:
    log γi = −A Zi2 (√I  /  (1+B √I )) + ∑ ε(i,j) mj

where I is the ionic strength; A is the Debye-Hückel slope (a solvent parameter); B is a temperature-dependent parameter:

t (°C) A B
0 0.4911.48
25 0.5091.50
50 0.5341.52
100 0.6001.56

and ε(i,j) is the specific interaction parameters between the ion “i” and another ion “j” of opposite charge sign. The summation is taken over all ions j such that Zi×Zj > 0 (opposite electric charge sign). Often the ε-values are assumed to be independent of the ionic strength. A fundamental principle of the SIT is that ε(i,j) = ε(j,i), that is, the specific ion interactions are symmetrical.

The activity of water (H2O) is given by:
    log aH2O = − Φ ∑mk / (ln(10) 55.508)

where “k” are all solute species in the aqeuous solution, and “Φ” is the osmotic coefficient, calculated using the SIT as follows:
    Φ = 1 − (2/3) (ln(10)/∑mk) A I3/2 σ(B √I) + (ln(10)/∑mk) ∑i ∑j ε(i,j) mi mj

where “i” are cations or neutral species and “j” are anions, and
    σ(x) = (3/x3) ((1+x) − 1/(1+x) − 2 ln(1+x))

Note that in the calculations the values of log γi are cut-off to be in the range ±(20 Zi2), and log aH2O is cut-off to be in the range ±4. Also the concentrations mi etc in all equations above are cut-off at 100 molal. The ionic strength is nevertheless calculated (I = ½ ∑ mi Zi2) without restrictions in the values of mi.

ε-values:

ε-values are first searched in a file named:

“SIT-coefficients.dta”

This file is distributed and stored in the installation folder. If you wish to add or change ε-coefficients you have to create a copy of the file (with the same name) and store it either in your user directory (system dependent, shown with the “Help / About” menu), or in the “working” directory (where your input data file is located). Any ε-values in the “working” directory will supersede those in the user directory, and the data in the user directory will supersede those in the installation directory.

If the first line in any of the three files described above is equal to “NoDefaults”, then all non-defined ε-values will be set to zero. Otherwise all non-defined ε-values will be set to the default values defined below.

  • Default ε-values between any ion and either Na+, Cl− or ClO4− are assigned with the estimation method proposed by Hummel (2009):

    ε(Na+,XY) = 0.05 Y

    ε(MZ,Cl−) = −0.05 + (0.1 Z)

    ε(MZ,ClO4−) = 0.2 Z

    where Z and Y are the electric charges for the cation (Z>0) and anion (Y<0), respectively. For example, ε(Sr2+,Cl−) = 0.15, and ε(Na+,MoO42−) = −0.1.

  • Finally, any ε-value not yet defined is set to:

    ε(MZ,XY) = 0.15 + 0.15 (Z + Y)

    where Z and Y are the electric charges for the cation (Z>0) and anion (Y<0), respectively. For example: ε(K+,MoO42−) = 0, and ε(H+,OH−) = 0.15.

Convergence problems

The nature of the SIT equation is such that in concentrated solutions the values of log γi become linearly dependent on the calculated concentrations. This may create convergence problems in the calculations, especially when making diagrams where the pH is large or small, and/or where the redox potential, Eh, is outside the stability range of H2O. If the redox potential is high, that large amounts of O2(aq) are calculated, or large amounts of H2(aq) if Eh is too low, and because of that ∑mk becomes unrealistically large.
