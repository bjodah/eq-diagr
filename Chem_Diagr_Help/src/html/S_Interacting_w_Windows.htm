SPANA: Interacting with Windows

The main window reacts to drag-and-drop:

  • Have both the File Manager (or Windows Explorer) and the main SPANA-window visible on the screen,
  • Select a file (or a group of files) in the File Manager,
  • Click the left mouse button down, and without releasing it, move the mouse (“drag” the file) away from the File Manager, place the cursor on top of the main SPANA-window, and then release the mouse button (“drop” the file).

This will have the same effect as selecting a file name by pulling down the “File” menu of the main SPANA-window.

File association: Files with extension “plt” and “dat” may be automatically associated with SPANA during setup. In such a case, a diagram is displayed when its name is double-clicked in the File Manager, and a data-file may be added to the file-name list when its name is double clicked. File association may be changed with the menu “Preferences / General options”.
