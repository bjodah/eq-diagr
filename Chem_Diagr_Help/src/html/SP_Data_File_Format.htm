SED & PREDOM

Input Data-File format

Input data files for SED and PREDOM have names ending with “dat”.

Data files are plain text files and may be edited with any text editor.

How to write Names and Numbers  in data files.

Structure of data files:

    1)  the chemical system
2)  diagram type and
3)  concentrations for each chemical component.

Title: The first line after the concentrations (if found, and not empty) is used as a title for the diagram. After that, any number of lines with comments may be added.

Several input file examples are found in subdirectory “.\EXAMPLES”.

Using DATABASE is the easiest way to create data files: from equilibrium constants available in an electronic database. The program SPANA may be used to modify the chemical system: remove components, interchange components for reaction products, deactivate species, etc.
