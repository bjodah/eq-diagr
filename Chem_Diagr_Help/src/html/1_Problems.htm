Problems when running DataBase/Spana?

What to do if you try the “Chemical Equilibrium Diagrams” sofware and things do not work?

There are two kinds of possible problems:

  • You can not run a program or there is an error. A message appears (or not). Perhaps the program terminates abruptly, or does not even start. These type of problems are discussed below.
  • The programs run but the results seem wrong:
    • Parts of the diagram are missing: the calculations failed
    • The diagram seems wrong
    • The equilibrium constants obtained from DataBase are erroneous: Modify a database

If the programs do not run, or you find an error

There might be programming errors or “bugs”, or Java exceptions. In such cases a special window opens atomatically, reporting the problem. Or the program(s) might be performing in non-intelligent ways. Please send these observations to the author at:

If a program does not start: before contacting the author open a terminal console and type:

java -jar program_name.jar

The message(s) that you get will give a clue to what the problem might be.

Because Spana calls background programs to make the calculations and to handle diagrams, there is always the possibility that these other programs do not work in your computer environment. To find out the source of a problem:

  • In Spana open the message window with the menu “Preferences / Show messages and errors”.
  • In the menu “Preferences / General options” select “advanced” program level. Then select “Verbose output of program messages”.
  • Then in the menu “Preferences / Calculation options” select
    • “Keep windows open after calculations”
    • “Write messages in SED & Predom”
    This will keep the calculation windows (SED or Predom) open. Any error messages reported in these windows may give a clue to how to solve the problem.
