Redox reactions:

Between components: The formation reactions for both redox species in a redox couple must be given. For example:

Fe 2+;13.02;;;Fe 3+;1;;;e-;1;;;;;;;0
Fe 3+;-13.02;;;Fe 2+;1;;;e-;-1;;;;;;;0

This allows DATABASE to find both Fe(II) when Fe 3+ is chosen as component, and to find Fe(III) when Fe 2+ is chosen. Note the sign (+ or −) for the formation equilibrium constant and for the stoichiometric coefficients.

Products of redox-reactions: for example, the formation of magnetite:

3 Fe2+ + 4 H2O 2 e− + 8 H+ + Fe3O4(s)

line in the database:

Fe3O4(s);-41.58;;;Fe 2+;3;H+;-8;e-;-2;H2O;4;;;;;-8
