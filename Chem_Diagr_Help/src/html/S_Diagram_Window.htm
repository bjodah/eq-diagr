The diagram window

It appears when SED/PREDOM finish a calculation. It is also created when a diagram is selected in the main window, or double-clicked in the File Manager / Explorer. Several diagram windows may be displayed simultaneously.

When the plot-file is read, it is displayed with the options (colours, font, etc) selected in the menu “Preferences / Diagram windows” of the main window. After that, the diagram is kept in memory for faster update when resizing, etc. If you make changes manually to the plot file with a text editor, e.g. to add, delete, or move labels, then you need to use the “MENU / Redraw (Ctrl-R)” in the diagram window to read the plot file again and to show the changes.

Displaying coordinates:

To read the coordinates of a point in a diagram, place the mouse on the point and click on the right mouse button: a small window appears with the coordinates at that point.
