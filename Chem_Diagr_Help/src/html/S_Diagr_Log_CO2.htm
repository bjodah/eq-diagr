Example of a logarithmic diagram showing the concentrations (on a logarithmic scale) for all species in the system H+, CO2(g). Note that the component CO32− in DATABASE has been exchanged for a reaction, in this case for CO2(g).

Note that the concentration of CO2(g) is not shown. The calculations consider the aqueous phase only, and the concentrations for gases are set to zero. The fugacity (≈ partial pressure) is calculated and it is displayed in log activity diagrams.
