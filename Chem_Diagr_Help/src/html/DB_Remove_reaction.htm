Remove a Reaction from the databases:

If you need to remove a reaction given in a text logK°-file, erase the corresponding line with a text editor.

If you wish to remove a reaction given in the binary logK°-database, so that it may not be retrieved, give again the name of the reaction product but preceede the name by a “@”. For example:

@FeOH+;

The order by which the database files are searched may be changed using the pull-down menu “Options / Data / Database files” of DATABASE. The order is important: for two species with the same name (but different logK° values), only the last one will be retained. To remove a species (by preceeding the name with a “@”) it is necessary that the species to remove does not appear in a database which is read later.

See also: Removing or Changing data.
