DATABASE

Databases

A database consists of two files:

  • A logK-file includes stoichiometry and logK° values for reactions.

  • An elements-file has information on which components should be listed under a given chemical element.

    For example, if the organic ligand glycinate is added to the database, the DATABASE program has no way of knowing that it should be listed under carbon. This information is given in the element-file.

Which databases? Any number of databases may be read by the DATABASE program. Use the pull-down menu “Options / Data / Database files” to add or remove databases.

The binary database supplied consists of two files “Reactions.elb” and “Reactions.db”. These files can not be edited.

What Order? The order in which the databases are searched may be changed using the pull-down menu “Options / Data / Database files” of DATABASE. The order is important because for two species with the same name (but different logK value), only the last one will be retained, see Removing or Changing data.
