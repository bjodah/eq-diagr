Temperature when making a data file for SPANA

In DATABASE the database files contain equilibrium constants at 25°C, and temperature extrapolations are possible if values are also available for the reaction enthalpies (and preferably also reaction heat capacitites).

Using the menu “Options / set Temperature” in DATABASE it is possible to select a temperature between 0 and 100°C. If a temperature, “t”, different than 25°C is selected, then the equilibrium constant is calculated using the van't Hoff equation:

log K°(T)  =  log K°(T0) + {ΔH° / (R ln(10))} × {(1/T0) − (1/T)}

where T and T0 are absolute temperatures in Kelvin (T0 = 298.15 K) and R is the gas constant. If a reaction heat capacity is also available, then the constant heat-capacity equation is used instead:

log K°(T)  =  log K°(T0) + {ΔH° / (R ln(10))} × {(1/T0) − (1/T)} + {ΔCp° / (R ln(10))} × {(T0/T) − 1 + ln(T/T0)}

When searching data in DATABASE an error message is displayed if enthalpy data is missing for any species, and an input data file for SPANA will then not be saved. You must then change the temperature back to 25°C to be able to save your search results.

Temperature when making a diagram in SPANA

DATABASE writes the selected temperature in the input data-file in a comment at the end of the first line, and this value is used when making a diagram if ionic strength effects are calculated, or if Eh is displayed in diagrams.
