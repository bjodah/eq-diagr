SED & PREDOM / Input data-file format

Data file structure: Plot definition

Input files consist of: 1) definition of the chemical system; 2) plot information; and 3) concentrations (ranges) for the chemical components.

The plot definition differs between SED and PREDOM:

Data to be entered for SED:

YCOMP, [Ymin, Ymax,] [EH,] XCOMP,

Data to be entered for PREDOM:

[EH,] [ONE AREA, NAME,]
YCOMP, XCOMP, MCOMP,

where data within [] are optional. A complete explanation is available for SED and PREDOM.

A list of possible diagram types may be obtained from SED or PREDOM by entering a “?” in the plot-definition input line, and running the program (this will make SED or PREDOM complain that “?” is not an acceptable diagram type, and the program will print a list of valid choices for the plot definition).
