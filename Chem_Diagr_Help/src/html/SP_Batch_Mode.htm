Running in Batch Mode

In many cases it is convenient to generate diagrams automatically, e.g. with a batch command file. All programs in the “Chemical Equilibrium Diagrams” package, for example SED and PREDOM may be invoked according to:
  java -jar SED.jar [data-file-name] [-arg=value]

SPANA may also be invoked as follows:
  java -jar Spana.jar [data-file-name] [-arg=value]
and DataBase:
  java -jar DataBase.jar [directory-name] [-dbg]

A list of available command-line arguments may be obtained from any of the programs (SED, PlotPDF, DataBase, etc) by giving the argument “-?”.

All programs will output extra (debugging) information by giving the argument “-dbg”.
