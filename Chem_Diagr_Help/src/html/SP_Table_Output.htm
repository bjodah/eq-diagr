SED & PREDOM

SED and Predom: Output of calculated values

Table output from SED

Output files can be generated by SED with the curves in the diagrams. These are plain text files that will get the same name as the plot-file, but with an extension selected by the user.

Table output can be obtained either by:

  • running from the command line or from a batch file
  • using SPANA: select first the “advanced” program level, and then the output file extension from the menu “Calculation options”. A table output file will be generated by checking the box when making diagrams.
You can use these files to insert the tabulated values into a spread-sheet (such as Excel), or you can plot the data with some other plotting program.

Output from Predom

Predom generates a list of points defining the boundaries between areas of predominance. This list is probably not useful to most users. Nevertheless, the list may be obtained either:

  • using the “-dbg -keep” command line parameters
  • through SPANA: select first the “advanced” program level, and then check the “Keep windows open” and the “Write messages” boxes in menu “Calculation options”.
In both cases a list of points defining the boundaries between areas of predominance may be obtained after the calculations are finished, and the diagram displayed, from the “Messages” tab of Predom. You may copy and paste the information, or use the menu “debug / save messages to file” in Predom to save the “Messages” tab to a text file.
