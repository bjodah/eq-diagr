A predominance area diagram shows the species formed with a given “main” component as a function of two variables. The following rules are used:

  • A solid is always shown on top of aqueous species, even when the solid has a very small concentration,
  • If more than one solid are formed, the solid with the largest fraction of the main component is shown,
  • If no solid is formed, the diagram shows the aqueous species which has the largest fraction of the main component.
