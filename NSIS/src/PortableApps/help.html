Portable Chemical Equilibrium Diagrams

Portable Spana/Database are the lightweight Chemical Equilibrium Diagrams Software (Java version) packaged with a “PortableApps.com” launcher as a portable app, so you can explore chemical equilibria in aqueous solutions from an USB flash drive, portable hard drive, CD or any other portable media. It has the same features as Spana/Database. Plus, it leaves no personal information behind on the machine you run it on, so you can take your chemical diagrams software wherever you go.

Portable Spana/Database - Specific Issues

The file “SpanaPortable.exe” is a launcher, that is, a small program that executes the Spana program itself. Database has a corresponding launcher.

These launchers are written using NSIS, and the source code is included in subdirectory “Other\Source”. The launchers have the following functions:

  • They check that the exe-files “Spana.exe” and “Database.exe” are where they should be. The directory where Spana/Database are located is given in the INI-files “SpanaPortable.ini” and “DatabasePortable.ini”, see the description in “Other\Source\Readme.txt”.

  • The subdirectory “Data\settings” can contain the file “Spana­Portable.reg” with saved Windows Registry data (file associations) from the last program execution. Database does not make any registry changes. When launching the Spana program, any existing Spana-data in the Windows Registry (from a local copy of the program) is saved as backup in the Registry itself. Then any data found in “Spana­Portable.reg” is inserted into the Windows Registry.

    Note that the drive letter can change when the removable drive is inserted into another computer. The file “Spana­Portable­Settings.ini” in subdirectory “Data\settings” contains the drive letter from which the Spana/Database launchers started during the last session. If the drive letter has changed since the last session, the launcher will search “Spana­Portable.reg” and the previous drive letter will be replaced with the current drive letter before copying the information to the Windows Registry. The same action is taken on the files “Spana.ini” and “Database.ini”.

    When Spana ends the data in the Windows Registry is saved in “Data\settings” and the backup information in the Registry is then restored, leaving the Registry in the same state as it was before launching SpanaPortable. The current drive letter is saved in file “Spana­Portable­Settings.ini”.

  • Finally “Spana.exe” (or “Database.exe”) is launched.

Note that any file name associations with Spana portable only operate while Spana protable is running. When you end Portable Spana the file associations will go back to those defined in the local computer.

Note that after starting the Spana launcher, the restoring of any Windows registy data will be made only after all Spana windows are closed, either local or portable. Unpredictable behaviour could result if a local copy of Spana is started while the portable Spana is running, or vice-versa (because of the shuffling of registry entries for file associations).
